The folder consists of:
a) example_hpn - two robotic systems specified utilising HPN - ready to read by the RSHPN Tool
b) gazebo_lwr4_simulator - model of lwr4 manipulator expressed in gazebo, use two scripts to compile and run the simulator
c) generated_ros_lwr4 - generated ROS code for LWR4 manipulator presented within the article - all files have been generated automatically by the RSHPN Tool
d) figat.mp4 - video presenting the RSHPN Tool

Note that the all necessary tutorials presenting how to use and navigate through above-mentioned projects will be available at: https://github.com/mfigat/public_rshpn_tool
